# ASP.NET Core Vulnerable Lab

SecurityTestingLab source-backed vulnerable web application lab.

Coverage: .NET/Kestrel/Razor/MVC route and model-binding coverage

Public upstream fallback reference: https://github.com/dotnet/AspNetCore.Docs

Runtime placement: windows-iis-target port 18200

No credentials or target-local generated values are stored in this source cache.

The source cache includes an IIS static fallback so the target remains reachable if a Windows Server has IIS but no .NET Hosting Bundle.
